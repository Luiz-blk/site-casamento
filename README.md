
💍 Site de Casamento – Luiz & Isabelly

Projeto de um site de casamento desenvolvido com foco em prática de HTML, CSS e JavaScript, com estrutura organizada e componentes reutilizáveis.

📌 Sobre o projeto

Este site foi criado para compartilhar informações importantes sobre o casamento, como:

Data do evento
Mensagem para os convidados
Página de confirmação de presença
Navegação dinâmica com menu reutilizável
🖥️ Funcionalidades
✅ Menu dinâmico carregado com fetch
✅ Layout organizado por páginas
✅ Estrutura modular (HTML, CSS e JS separados)
✅ Menu responsivo com botão hambúrguer
✅ Interação com JavaScript (DOM e eventos)
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript (ES6)
