## 📌 Sobre o Projeto

Este site foi desenvolvido com foco em prática de **HTML, CSS e JavaScript**, com estrutura organizada e componentes reutilizáveis. Reúne tudo o que os convidados precisam saber sobre o casamento em um só lugar.

---

## ✨ Funcionalidades

- 📅 Informações sobre data, local e horário do evento
- 💌 Mensagem personalizada para os convidados
- ✅ Confirmação de presença (RSVP)
- 🍔 Menu responsivo com botão hambúrguer
- 🔄 Menu dinâmico carregado via `fetch`

---

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura das páginas |
| CSS3 | Estilização e responsividade |

## 📂 Estrutura do Projeto

```
site-casamento/
├── assets/

│   ├── js/         # Scripts JavaScript
│   └── img/        # Imagens e recursos visuais
│
├── index.html          # Página inicial
├── informacoes.html    # Informações do evento
├── convidados.html     # Confirmação de presença
└── menu.html           # Componente de navegação
```

## ▶️ Como Executar

**1. Clone o repositório**
```bash
git clone https://github.com/luizsansilv/site-casamento
```

**2. Abra o projeto**

- Diretamente no navegador via `index.html`, ou
- Utilize a extensão **Live Server** no VS Code para hot reload
